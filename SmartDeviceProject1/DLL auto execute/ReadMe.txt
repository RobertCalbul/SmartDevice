AppWizard ha creado esta aplicación DLL auto execute.  

Este archivo contiene un resumen de todo lo que encontrará en los archivos que
constituyen la aplicación DLL auto execute.


DLL auto execute.vcproj
    Éste es el archivo de proyecto principal para los proyectos de VC++ generados mediante un Asistente para aplicaciones. 
    Contiene información acerca de la versión de Visual C++ que generó el archivo e 
    información acerca de las plataformas, configuraciones y características del proyecto seleccionadas con el
    Asistente para aplicaciones.

DLL auto execute.cpp
    Ésta es la aplicación principal del archivo de código fuente.

/////////////////////////////////////////////////////////////////////////////
AppWizard ha creado los siguientes recursos:


DLL auto executeppc.rc
    Ésta es una lista de todos los recursos de Microsoft Windows que
    utiliza el programa al compilar la plataforma Pocket PC 2003, o una plataforma
    compatible con el mismo modelo de interfaz de usuario. Incluye los iconos, mapas de bits 
    y cursores almacenados en el subdirectorio RES.  Este archivo puede 
    editarse directamente en Microsoft Visual C++. Cuando se almacena el archivo .rc, 
    se almacenan las definiciones de la sección de datos como versión hexadecimal
    del valor numérico para el que se han definido, en lugar de como nombre descriptivo de
    la definición.

DLLautoexecuteppc.rc2
    Este archivo incluye recursos no editados por Microsoft
    Visual C++.  Se deberían colocar todos los recursos que no pueden editarse mediante el
    editor de recursos en este archivo.

Resourceppc.h
    Éste es el archivo de encabezado estándar, que define nuevos identificadores de recurso.
    Microsoft Visual C++ lee y actualiza este archivo.




DLL auto execute.ico
    Éste es un archivo de icono, que se utiliza como el icono de la aplicación (32x32).
    Este icono está incluido en el archivo de recursos principal DLL auto execute.rc.

/////////////////////////////////////////////////////////////////////////////
Otros archivos estándar:

StdAfx.h, StdAfx.cpp
    Estos archivos se utilizan para generar un archivo de encabezado precompilado (PCH)
    denominado DLL auto execute.pch y un archivo de tipos precompilado denominado StdAfx.obj.
